Image Upload Widget is a widget that allows users to upload images.

== Description ==

Image Upload Widget is a simple widget to upload an image that uses the native WordPress media manager to upload.

* Link the image
* Title
* All fields are optional.
* Responsive

== Installation ==

1. Upload the folder 'image-upload-widget' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
